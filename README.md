# Receipt Intelligence System

A modular receipt parsing and intelligence system that converts OCR receipts and email HTML into structured datasets using Regex, XPath, Flask APIs, and SQLite.

## Features

- OCR receipt cleaning
- Regex extraction engine
- XPath email parsing
- Fallback parsing logic
- REST APIs with Flask
- SQLite database storage

## Run

python api/app.py

## API Endpoint

POST /parse_receipt

## Technologies

Python, Flask, Regex, XPath, SQLite
